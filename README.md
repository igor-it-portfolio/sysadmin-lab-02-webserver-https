# Servidor Web com HTTPS (Nginx)

Este projeto demonstra a configuração de um servidor web Linux
utilizando Nginx com HTTPS, aplicando boas práticas de rede,
segurança e documentação.

## Objetivo
Configurar um servidor web funcional e seguro, acessível via HTTPS,
simulando um cenário real de produção.

## O que foi implementado
- Instalação e configuração do Nginx
- Criação de Virtual Host
- Configuração de HTTPS com Let's Encrypt
- Testes de conectividade e portas
- Organização profissional de arquivos

## Estrutura do repositório
configs/
  nginx.conf
  site.conf
docs/
  README.md

## Tecnologias utilizadas
- Linux (Ubuntu Server)
- Nginx
- OpenSSL / Let's Encrypt
- Git & GitHub
